# Local Chat UI Prototype

A simple, polished frontend for a location-based temporary chat room web app. Built with only HTML, CSS, and vanilla JavaScript.

## Features
- Minimalist, soft, and calm design aesthetic.
- Pure CSS styling utilizing variables and modern constraints.
- Mobile-first layout that centers nicely on larger screens.
- Interaction mocking via `localStorage` and vanilla JavaScript events.

## Structure
- `index.html`: The entry layout where a user enters their name.
- `rooms.html`: A discovery view showcasing nearby temporary chat spaces, complete with a modal for creation.
- `chat.html`: The active messaging interface with automated dummy replies.

## Usage
Simply open `index.html` in any modern web browser to interact with the prototype. No dev server or build process is required.
